HTML Requirements

  1. Appropriate HTML5 tags, including use of HTML5 structural tags, <a> , <p> , <ol> , <ul> , and <dl> , to name a few
  2. Inline images both static and clickable
  3. Use of appropriate special characters
  4. Use of appropriate <meta> tags
  5. A footer on the home page indicating when the website was last updated or modified
  6. Use of appropriate form elements including some of the new HTML5 form elements
  7. Apply appropriate styles to web forms
  8. Apply inline form validation by using the focus pseudo-class to appropriate form elements.
  9. Use sound and video somewhere on your site. Ensure these are of short duration, preferably about 30 seconds.
  10. Use of transitions and animations

CSS Requirements

  1. Use CSS3 to format the appearance of paragraphs, lists, headers, background images, and forms
  2. Use CSS3 contextual selectors ( e1, e2, e3, e f; e > f; e + f) and at least two attribute selectors, excluding class and id attribute selectors
  3. Use of CSS3 styles for color ( RGBA & HSLA )
  4. Use of dynamic pseudo-classes to highlight hyperlinks states of link, visited, active, hover; focus and two structural pseudo-classes: first-of-type and last-of-type
  5. Use of CSS3 background styles ( background size, background-clip, background-origin, padding, margins, borders, use of an appropriate border style, use of rounded corners)
  6. Your home page should have a drop cap created with CSS3
  7. Define document content with the class and id properties and create styles for them
  8. Use each of the following page layout techniques: Grid Flex Float Positioning
  9. Use of CSS3 text shadows and box shadows
  10. Use of CSS3 transform rotate feature
  11. Use of CSS3 opacity applied to two images to appear semi-transparent
  12. Your website design should include responsive design implementation using concepts covered in this course that would accommodate desktop, tablet, and mobile designs, including positioning, grid, and flex layouts
  13. The use of contextual selectors to control sub level menus is required, including a navicon image for mobile design
  14. Images should also be responsive for different screen sizes and devices

JS Requirements

  1. Use of variables, arithmetic, comparison and logical operators, subroutines and functions, conditional statements, conditional operators, and running time commands with system clock
  2. Use of system date (example: clock, countdown timer to an event) on your website
  3. Validating form input by using appropriate JavaScript functions
  4. Use of JavaScript alerts for validation purposes
